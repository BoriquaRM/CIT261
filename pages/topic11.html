Example 1 : Changing the background color when the cursor hovers over it.

CSS CODE


.transition1 {

padding: 5px;
background-color: red;
min-height: 50px;
/*transition: all 2s ease;*/ }

.transition1:hover, .transition2:hover {

background-color: green; }
The :hover tag in CSS will affect the object when the cursor "hovers" over it. In this case, it changes color from red to green.
(Note: transition is disabled)
2
Example 2 : Animating the changing background to gradually shift color when the cursor hovers over it.

CSS CODE


.transition2 {

padding: 5px;
background-color: red;
min-height: 50px;
transition: all 2s ease; }

.transition2:hover {

background-color: green; }
The :hover tag is still in effect, but the transition tag in CSS is now enabled, giving the intended effect over 2 seconds.
3
Example 3 : Combining these concepts into grouped <div>'s.

CSS CODE


.transition1, .transition2 {

margin: 9px;
padding: 5px;
background-color: red;
min-height: 50px;
/*transition: all 2s ease;*/ }

.transition1:hover, .transition2:hover {

background-color: green; }

.transition2 {

transition: all 2s ease; }

TOP: Example 1 is the outside border, with Example 2 as the center.

MIDDLE: Now Example 2 is the outside border, and Example 1 is the center.

BOTTOM: A doubled-up version of the TOP, but one of the border layers is Example 2.
4
Example 4 : Using images.

CSS CODE



#transImg {

position:relative;
height:281px;
width:450px;}

#transImg img {

position:absolute;
transition: opacity 1s ease-in-out;}

#transImg img.imgTop:hover {

opacity:0;}

.imgTop, .imgBottom {

width: 200px; }
Using a <div> with <img>'s inside, two pictures are layered one on top of the other, but the .imgTop is dissolved away once the mouse hovers over it, showing the .imgBottom picture. In this way, the image transitions from one to another.
