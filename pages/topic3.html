Topic 3 - JSON Parse & Stringify

INDEX

  1. Parse
  2. Stringify
1
Parse : The exchange of data to/from a web server. When receiving data from a web server, the data is always a string.

JAVASCRIPT CODE


var content = '{"name1":"David", "name2":"Olsen", "age":33, "city":"Phoenix", "state":"AZ", "food":"Nachos"}' var obj = JSON.parse(content); document.getElementById("pID1").innerHTML = "Hello, my name is " + obj.name1 + " " + obj.name2 + ", I am " + obj.age + " years old, I live in " + obj.city + ", " + obj.state + ", and my favorite food is " + obj.food + ".";

The .parse property allows JavaScript to select from the array of information within the internal JSON code in the variable content. The information is called up within its own hierarchy by placing a period between obj and the information desired.
2
Stringify : Returns JSON information as a string, instead of the values of the properties.

JAVASCRIPT CODE


var obj2 = {"name1":"David", "name2":"Olsen", "age":33, "city":"Phoenix", "state":"AZ", "food":"Nachos"}; var myJSON = JSON.stringify(obj2); document.getElementById("pID2").innerHTML = myJSON;

The .stringify property takes the entire string of information within the JSON, wihout removing the syntax of property names. This method of displaying information is not recommended.
