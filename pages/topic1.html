1.1
Loops : Repeatedly runs a block of code, until a specific condition is met. Uses for, for/in, while and do/while. Loops are considered the most basic of JS applications.

for : loops through a block of code a set number of times, often utilizing an increasing/decreasing tag such as i++ or i--.

JAVASCRIPT CODE


function for1() {
document.getElementById("pID1aa").innerHTML = "99 Bottles Song";
var content = "";
var x;
for (x = 99; x > 96; x--) {
content += "<b>" + x + "</b> bottles of beer on the wall,<br><b>" + x + "</b> bottles of beer!<br>Take one down, pass it around,<br><i>" + [x-1] + "</i> bottles of beer on the wall!<br>";
document.getElementById("pID1a").innerHTML = content;
}
document.getElementById("buttonID1a").style = "display:none;";
document.getElementById("pID1ab").innerHTML = "And so on...";
}


This for loop runs a function, as long as the conditions remain true. The variable x begins at 99, runs as long as it is greater than 96, and decreases once the function finishes.

To include the decreased variable in the same function operation, I added [x-1] to update the x variable, and begin with it in the next iteration.

1.2
for/in : loops through the properties of an object.

JAVASCRIPT CODE


function forIn1() {
var content = "";
var person = {name1: "David", name2: "Olsen"};
var age = 33;
var x;
for (x in person) {
content += person[x] + " ";
document.getElementById("pID1b").innerHTML = "My name is " + content + ", and I am " + age + " years old.";
}
document.getElementById("buttonID1b").style = "display:none;";
}

This loop adds a " " between the name variables, so that they can be added into the final result. This way a space does not need to be added to each variable itself.

1.4
while : loops through a block of code while a specified condition is true

JAVASCRIPT CODE


function while1() {
var x = 0;
while(x < 8) {
var showLoop = document.getElementById('pID1');
showLoop.innerHTML += "Loop " + x + "." + "<br>";
x++;
}
document.getElementById("buttonID1").style = "display:none;";
}

This loop has a limit of 8, then terminates. It uses the "while" tag, indicating that while x < 4, it will continue running the function.

To ensure that the function does not repeat, the button is disabled once finished. Since JS considers "0" as the first number, so counts it as #1.

do/while : loops through a block of code while a specified condition is true.

JAVASCRIPT CODE


function doWhile1() {
var text = ""
var i = 0;
do {
text += "The number is " + i + "<br>";
i++;
}
while (i < 6);
document.getElementById("pID1e").innerHTML = text;
document.getElementById("buttonID1e").style = "display:none;";
}

A similar approach to the for function, in that it will continue to run a function while a certain set of conditions are true. In this case it "does" something "while" i is greater than 6.

Functions : A block of code designed to perform a particular task when called upon, identified with ().

JAVASCRIPT CODE


//Range of numbers to pick from
var pick = {
range: 1000,
roll: function () {
var randomNumber = Math.floor(Math.random() * this.range) + 1;
return randomNumber;
}
}
//Shows the number generated
function showNumber(number) {
var numberResult = document.getElementById('numberResult');
numberResult.innerHTML = number;
}
//Clickable button
var button = document.getElementById('button3');
//Clicking button runs "function()"
button.onclick = function() {
var result = pick.roll();
showNumber(result);
}

Clicking the button repeatedly will give a randomly chosen number, based on the range given inside the JS code.

The button does not disappear once clicked, making this function a repeatable one that replaces each number with a new one.

Associative Arrays : Arrays that the user can redefine and update as needed. Combinations incluse .shift, .unshift, .pop and .push.

JAVASCRIPT CODE


var foodPop = ["Chips", " Salsa", " Nachos", " Cake", " Cookies", " Chocolate"];
document.getElementById("pID7a").innerHTML = foodPop;
function arrayAssA() {
foodPop.pop();
document.getElementById("pID7a").innerHTML = foodPop;
if (foodPop.length < 1 || foodPop == undefined) {
document.getElementById("pID7a").innerHTML = "All gone.";
}
}
var foodShift = ["Chips", " Salsa", " Nachos", " Cake", " Cookies", " Chocolate"];
document.getElementById("pID7b").innerHTML = foodShift;
function arrayAssB() {
foodShift.shift();
document.getElementById("pID7b").innerHTML = foodShift;
if (foodShift.length < 1 || foodShift == undefined) {
document.getElementById("pID7b").innerHTML = "All gone.";
}
}


Clicking the top button removes the last item on the array, using the .pop property, until all of the items are gone.

Clicking the bottom button removes from the first array item, using the .shift property instead.

The button does not 'need' to get removed in this case, since the function has reach the end of its line and cannot repeat anyways.
