Loops : Repeatedly runs a block of code, until a specific condition is met. Uses for, for/in, while and do/while. Loops are considered the most basic of JS applications.
 JAVASCRIPT CODE

function loopFun() {
var x = 0;
while(x < 4) {
var showLoop = document.getElementById('loopID');
showLoop.innerHTML += "Loop " + x + "." + "
";
i++;
}
document.getElementById("buttonLoopID").style = "showLoop:none;";
}
This loop has a limit of 4, then terminates. It uses the "while" tag, indicating that while x < 4, it will continue running the function.
To ensure that the function does not repeat, the button is disabled once finished. Since JS considers "0" as the first number, so counts it as #1.
2
Conditional Statements : Executes a block of code if a specified condition is true, or another if false. Also known as If/Else statements.
(Similar to "boolean" assignments, which view results as True/False)
 JAVASCRIPT CODE

function function2()
{ var response;
var user = prompt("What's your name?");
if (confirm && user == "") { response = "Come on, you've gotta have a name."; } else if (user == "asdf" || user == 123) {
response = "Dude, '" + user + "' is not a name.";
}
else if (user == "Yoda") {
response = "Mmmm, Jedi master, you are.";
}
else if (user == null || user == "") {
response = "Well fine then...";
}
else {
response = "Hello " + user + "!";
}
document.getElementById("p2ID").innerHTML = response; }

Clicking the button will invite a prompt, where you can give a response. If something is typed in, then you will a proper greeting... unless you put in something specific.
Also, if you don't put anything in, the program will respond appropriately.
3
Functions : A block of code designed to perform a particular task when called upon, identified with ().
 JAVASCRIPT CODE

//Range of numbers to pick from
var pick = {
range: 1000,
roll: function () {
var randomNumber = Math.floor(Math.random() * this.range) + 1;
return randomNumber;
}
}
//Shows the number generated
function showNumber(number) {
var numberResult = document.getElementById('numberResult');
numberResult.innerHTML = number;
}
//Clickable button
var button = document.getElementById('button3');
//Clicking button runs "function()"
button.onclick = function() {
var result = pick.roll();
showNumber(result);
}

Clicking the button repeatedly will give a randomly chosen number, based on the range given inside the JS code.
The button does not disappear once clicked, making this function a repeatable one that replaces each number with a new one.
4
Variables : A storage location of information, unique to task. Declared by using var =.
 JAVASCRIPT CODE

// var a = 5;
// var b = "Five";
var a = 5;
var b = "Five";		 The variables here, a and b, both show how properties can be assigned as numbers or as strings. If numbers, then they do not require quotations. If written words, or strings, then they must be covered with double-quotations.

Parameters : Names listed in the function definition. Functions have () at the end of their name, but variables can be placed inside to define which ones are affected.
 JAVASCRIPT CODE

function theAnswer() {

The function name contains the variables x and y, which are defined within the function itself. When the button is clicked, it adds them together.

7
Associative Arrays : Arrays that the user can redefine as needed. Combinations incluse shift, unshift, pop and push.
 JAVASCRIPT CODE

var foodPop = ["Chips", " Salsa", " Nachos", " Cake", " Cookies", " Chocolate"];
document.getElementById("pID7a").innerHTML = foodPop;
function arrayAssA() {
foodPop.pop();
document.getElementById("pID7a").innerHTML = foodPop;
if (foodPop.length < 1 || foodPop == undefined) {
document.getElementById("pID7a").innerHTML = "All gone.";
}
}
var foodShift = ["Chips", " Salsa", " Nachos", " Cake", " Cookies", " Chocolate"];
document.getElementById("pID7b").innerHTML = foodShift;
function arrayAssB() {
foodShift.shift();
document.getElementById("pID7b").innerHTML = foodShift;
if (foodShift.length < 1 || foodShift == undefined) {
document.getElementById("pID7b").innerHTML = "All gone.";
}
}


Clicking the top button removes the last item on the array, using the ".pop" addition, until all of the items are gone.
Clicking the bottom button removes from the first array item.
