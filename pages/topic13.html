Topic 13 - CSS3 Transforms

1
Color Change : Rotates through colors on an animation sequence called @keyframes, after giving the keyframe sequence a unique name. Starts on a 3-second delay.
 CSS CODE

.animate1 {

background-color: red;
animation-name: example1;
animation-duration: 7s;}

@keyframes example1 {

0% {background-color: red;}
25% {background-color: yellow;}
75% {background-color: blue;}
100% {background-color: green;}}

This changes color

 This cycles through a set of colors in CSS, each changing at set percentages, treating it as a timeline. As a default, once it's completed it does not repeat.

(Note: the animation name needs to be unique from any other animations you have on the page, otherwise they may run the same animation)
2
Moving the Object : This moves the object on a set path of motion, then returns it to its starting point. Uses the animation-iteration-count: 2 tag.
 CSS CODE

.animate2 {

padding: 10px;
background-color: red;
position: relative;
animation-name: example2;
animation-duration: 2s;
animation-delay: 9s;
animation-iteration-count: 2; }

@keyframes example2 {

0% {left:0px; top:0px;}
25% {left:400px; top:0px;}
50% {left:200px; top:200px;}
75% {left:0px; top:-300px;}
90% {left: 0px; top: 50px;}
100% {left:0px; top:0px;}}

This moves around

 This animation starts 9 seconds after the page loads, and repeats twice. The @keyframe tags control the X&Y coordinates at the percentage points, much like a timeline.

The position: relative; tag is vital so that the animation occurs where it's placed.
3
Forever Chain : This is a chain of animations that goes on without end. Uses the : infinite tag.
 CSS CODE

.animate3 {

padding: 10px;
background-color: red;
position: relative;
animation-name: example3;
animation-duration: 3s;
animation-iteration-count: infinite;}

@keyframes example3 {

0% {left:0px; top:0px; }
10% {left:0px; top:0px;}
20% {left:0px; top:5px; transform: rotate(0deg);}
80% {left:0px; top:50px; transform: rotate(30deg);}
100% {left:0px; top:0px;}}

This goes on forever...

This animation goes on forever, because of the : infinite tag, but it also incorporates the transform: rotate tag from Topic 12.

(Note: the rotate animation is set at 0 on the 20% marker, meaning that it will start rotating after 20%)
4
Hover Animation : Utilizing the :hover tag to trigger an animation.
 CSS CODE

.animate4 {

padding: 10px;
background-color: red;}

.animate4:hover {

position: relative;
animation-name: example4;
animation-duration: 3s;
animation-iteration-count: infinite;}

@keyframes example4 {

0% {left:0px; top:0px; }
50% {left:50px; top:0px;}
100% {left:200px; top:5px; transform: rotate(90deg);}}

Hover over me!

When the mouse is placed over the red box, the animation will start, but will end when the mouse is no longer hovering on it.
5
Many Examples : Utilizing previous tags into a single animation.
 CSS CODE

.animate5 {

padding: 10px;
background-color: red;
position: relative;
animation-name: example5;
animation-duration: 8s;
animation-iteration-count: infinite;}

@keyframes example5 {

0% {left:0px; top:0px; }
30% {left:10px; top:-90px; transform: skewX(60deg);}
70% {left:160px; top:50px; transform: rotate(181deg);}
90% {left:160px; top:50px; transform: scale(2);}
100% {left:0px; top:0px; transform: rotate(0deg); transform: scale(1); transform: skew(0deg);}}

I feel sick...

Using rotate, scale, skew and move. Only one transform tag can be used in a line.
