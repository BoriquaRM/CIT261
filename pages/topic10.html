Video : Inserting a video file into HTML.
 HTML CODE

<video width="350" controls>
<source src="../images/TheHunterDay.mov" type="video/quicktime">
</video>
The <video> tag inserts a video file that's saved within the website's own database. Must include the controls tagline to use any of the interactive controls. By default, the size of the video will be whatever its pixel dimensions are. To resize this, set the width within the tagline.
( Fun Fact: This video is an art proposal I designed for the University of Wisconsin, entitled "The Hunter" )
1.1
iframe : Same concept, but this takes a video from another website and embeds it into the website.
(Video taken from YouTube: see it here)
 HTML CODE

<iframe width="350" src="https://www.youtube.com/embed/B7T4f9nkAFM?controls=0" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
The <iframe> tag links a video from another website, a popular choice being YouTube. However, the link may not always work, or the video could get pulled from YouTube, potentially leaving the video unavailable.
( Can easily be obtained from YouTube videos by clicking Share > Embed > Copy )
2
Audio : Inserts a tab similar for interacting with a video, but without a screen. This is just for playing sounds / music. HTML CODE

<audio controls><source src="../images/dialup.mp3" type="audio/mpeg"></audio>
The <audio> tag here plays the audio file that is included in the website's own database. Must include "controls" in the tag. All of the details take care of themselves, such as Play controls, the length of the sound, and volume control.

Canvas : Inserts a space for colored areas using JavaScript. HTML CODE

<canvas id="canvasEx"></canvas>

JAVASCRIPT CODE

var c = document.getElementById("canvasEx");
var ctx = c.getContext("2d");
ctx.rotate(20 * Math.PI / 180);
ctx.shadowBlur = 10;
ctx.shadowColor = "black";
ctx.fillStyle = "#FF0000";
ctx.fillRect(60, 0, 150, 50);
The <canvas> tag inserts a space for JavaScipt to draw shapes, insert images, or otherwise create content, but must include JavaScript.
ctx.rotate : rotates the object
ctx.shadowBlur : creates a shadow blur
ctx.shadowColor : defines the shadow color
ctx.fillStyle : defines the object color
ctx.fillRect : defines the (x, y, width, height)
