Topic 12 - CSS3 Transforms

INDEX

  1. Rotating
  2. Skewing
  3. Scaling
1
Rotating : Tilting the box and text area on a desired degree amount. Uses the transform: rotate tag.
 CSS CODE

.transform {

background-color: yellow;
transform: rotate(30deg); }

This is rotated at 30°

 This does nothing more than rotate the existing object on a pivot point by 30°. The degree amount can be changed in CSS.
2
Skewing : This stretches the object along a single axis point, giving it a "leaning" look. Uses the transform: skew tag.
 CSS CODE

.skew {

background-color: orange;
transform: skewX(-30deg); }

This is skewed at -30° on the X-axis

 This leans the object at an angle, so it looks like it's leaning, trying to give a 3D concept. In this example, the object is leaning at a negative value.
3
Scaling : Stretches an entire object as a whole, or along the X/Y axis. It can give a "rubberband" effect, or a "squished" effect. Uses the transform: scale tag.
 CSS CODE

.scale {

background-color: red;
color: white;
transform: scaleY(0.4); }

This is scaled at .4 on the Y-axis

This example is a bit hard to read, only because it is squished at .4, which means it is 40% of its normal height. If the scaleY was set to 1.4, it would be stretched to 140% of its height.
