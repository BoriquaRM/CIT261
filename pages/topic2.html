Object Creation : The legs by which JavaScript stands. Variables, functions, properties, etc. all have to do with creating an object, or something for the user to see.

JAVASCRIPT CODE


function objFunction() { var user = { fName : String(document.getElementById("fName").value), lName : String(document.getElementById("lName").value), age : String(document.getElementById("age").value), food : String(document.getElementById("food").value), allName : function() { return this.fName + " " + this.lName; } } document.getElementById("pID1").innerHTML = user.allName() + " is " + user.age + " years old, and enjoys eating " + user.food + "."; }

2
Inheritance : the application of variables, and the values associated with them.

JAVASCRIPT CODE


function inheritanceFunction() { var user = new Object(); user.first = document.getElementById("name1").value; user.last = document.getElementById("name2").value; document.getElementById("pID2").innerHTML = "Hello " + user.first + " " + user.last + "."; }
First Name:
Last Name:

Like with normal object creation, this takes the variables inside the JS code, and applies them to a result. They inherit a value, and express it when called upon.
3
Properties : are the values given to variables, either as numbers or strings.

JAVASCRIPT CODE


user.first = "Charley";
user.last = "the Unicorn";
user.first = "Charley";
user.last = "the Unicorn";

 When identifying a property in JS, it is called up by naming the variable. In some cases, it is identified in the code itself, or offers the user a chance to submit their own property.
4
Methods : The manner in which a result is achieved. Calling up properties/values in a certain way, depending on the needs of the function.

JAVASCRIPT CODE


function methodFunction1() { var user = new Object(); user.first = "Charley"; user.last = "the Unicorn"; document.getElementById("pID4a").innerHTML = "Hello " + user.first + " " + user.last + " 1."; }
function methodFunction2() { var user = new Object(); user.first = "Charley"; user.last = "the Unicorn"; document.getElementById("pID4b").innerHTML = "Hello " + user["first"] + " " + user["last"] + " 2."; }


In the example of properties, the same code is used, but is called upon in a different manner. Instead of using .property, it uses ["property"].
( Typically, the .property style is the most commonly used )
