Topic 2 - JavaScript again

Definitions

  1. Object Creation Functions : The basis for understanding JS.
  2. Inheritance : Uses "constructors" to connect object in a chain of information.
  3. Properties : Giving an assigned value to a variable, by creating a new name, such as "person.age".
  4. Methods : A property containing a function definition.
  5. Instantiation : Ways to create something in JavaScript, consisting of Functional, Functional-shared, Prototypal and Pseudoclassical.
