Topic 7 - Editing CSS Properties Using JS

INDEX

  1. Manipulate CSS with JS
1
Manipulate CSS with JS : Instead of setting an absolute CSS value to objects, they can be manipulated by JavaScript.

CSS CODE


function mouseOn() { document.getElementById("imgTDjs").style.border = "2px solid red"; } function mouseOff() { document.getElementById("imgTDjs").style.border = "none"; } function changeSize() { document.getElementById("imgTDjs").style.width = "180%"; } function changeBorder() { document.getElementById("imgTDjs").style.borderRadius = "50px"; } function normal() { document.getElementById("imgTDjs").style.width = "77%" document.getElementById("imgTDjs").style.border = "none" document.getElementById("imgTDjs").style.borderRadius = "0px"; }
The image here can be manipulated by the buttons provided. The property values in CSS are altered to change the size, borders, and whatever else may be desired.

This can also apply to a mouse-over. It changes the properties when hovering over, and hovering off.
