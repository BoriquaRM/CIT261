Topic 6 - DOM Manipulation

INDEX

  1. createElement, appendChild, insertBefore, removeChild
1
createElement : Creates a tag in JavaScript, in the same way that HTML would create a <p> tag. This creates the elements when needed/called upon, so they do not need to take up excess HTML space.
appendChild : Terminates the created element, in the same way that an ending </p> tag would be placed.
insertBefore : Inserts an element before the one that is already in place.
removeChild : Removes a child element from a list, based on the array number indicated.

JAVASCRIPT CODE


function createFunction() { var liElement = document.createElement("li"); var text = document.createTextNode("Thing"); liElement.appendChild(text); var list = document.getElementById("pID1"); list.insertBefore(liElement, list.childNodes[0]); } function removeFunction() { var list = document.getElementById("pID1"); list.removeChild(list.childNodes[0]); }
    		This creates a new element, <li>, within a <ol> tagline. Each time you click Add, and removes the last element when Remove is clicked.
