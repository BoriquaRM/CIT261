Topic 5 - Local Storage API

  1. Local Storage API

Local Storage API : Creates an internal storage place of information within the webpage, typically base on the user prompts and input. Common uses are for cases in which the user needs to type in any kind of information that will be applied to a purpose in the website. (Comments, FAQ submissions, posts, input windows, etc.)
